## Quotas and Limits

Use this reference when the user is working with Chroma Cloud and you need to reason about request, collection, or account limits.

Do not rely on memorized values. Chroma Cloud quotas can change, and exact numbers should be verified against the official quotas page before making implementation decisions.

The official quotas page covers limits for categories such as:

- document and metadata sizes
- embedding dimensions
- query result counts
- filter and predicate limits
- collection metadata limits
- collection and record counts
- read and write concurrency
- other Cloud-specific quotas

Official documentation:
https://docs.trychroma.com/cloud/quotas-limits
