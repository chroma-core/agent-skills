## Local Chroma

For python, chroma can be installed with pip or uv

```
pip install chromadb
uv install chromadb
```
To run chroma in python from a venv:

```
chroma run
```

For javascript, npm can be used

```
npm install chromadb
```

To run chroma from where it was installed:

```
npx chroma run
```
